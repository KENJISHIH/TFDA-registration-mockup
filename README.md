# 化粧品產品登錄平台改版建議

> 臺中市化粧品工業同業公會 · 對 TFDA 115 年度化粧品產品登錄平台改版需求調查之建議

## 提案內容

涵蓋 **4 大改版主軸**、**16 項功能提案**、**8 項 API-Ready 架構原則**：

- **A 章 現有功能優化**（8 個 mockup）：全成分登錄、GMP 證書、製造場所、登錄表單、首頁儀表板、案件狀態、展延通知、成分品質
- **B 章 政策連動提案**（4 個 mockup）：電子標籤、品名風險預警、SA 數位平台、代工資訊透明化
- **C 章 風險管理與國際警訊**（4 個 mockup）：合規檢核、國際警訊、香料過敏原、風險趨勢儀表板
- **D 章 API-Ready 架構原則**（8 項）：永久 ID、欄位 Schema、內部 Dogfooding、OAuth、事件驅動、國際編碼、PIF 機器可讀、OpenAPI

## 開啟方式

直接開啟 `index.html`，或瀏覽部署版本。
